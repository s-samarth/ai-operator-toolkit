# The Operator's Guide to AI Tools

An interactive, military-operations-themed website that teaches AI tool selection and context management through tactical analogies. What if Claude Code was a weapon, Google AI Pro was a Swiss Army knife, and ChatGPT was a flashlight?

## About

This site maps AI coding tools to special operations equipment, using the metaphor of carry weight (context windows), ammo supply (subscription tiers), and marksmanship (prompt quality) to explain how to use Claude, Google AI Pro, and ChatGPT effectively together.

## Tech Stack

- React 18 + TypeScript
- Vite (build tool + dev server)
- Tailwind CSS + shadcn/ui (Radix UI primitives)
- Framer Motion (scroll & interaction animations)
- Lucide React (icons)

## Sections

1. **Hero** — Cinematic tactical intro with particle effects and scan-line HUD overlay
2. **The Carry Weight Rule** — Context window as equipment weight gauge
3. **The Armory** — Interactive weapon wall with 8 clickable equipment items
4. **Operator Tiers** — Free through Max 20x subscription comparison
5. **The Paradox** — Pro vs Max 20x operator showdown (tale-of-the-tape style)
6. **Three Layers of Skill** — Resource management, marksmanship, field craft
7. **Mission Loadouts** — 4 classified mission dossiers with gear recommendations
8. **The Three Kits** — Claude (weapon with rotating attachments), Google AI Pro (unfolding Swiss Army knife), ChatGPT (flashlight that turns on)
9. **The Doctrine** — 5 rules of engagement + 6 anti-patterns
10. **Essential vs Luxury Gear** — What to always carry vs mission-dependent
11. **Field Operations** — Session management tactics (/compact, /clear, Esc Esc, plan mode, /model)
12. **Closing** — Three-pillar doctrine summary + final quote
